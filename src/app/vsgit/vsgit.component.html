• Visual Studio Code (VS Code): A lightweight and customizable code editor that simplifies coding with its extensions and Git integration, along with debugging tools and a built-in terminal.
  • Git: A system that tracks and manages code changes, enabling teamwork, experimenting with new features using branches, and reverting to previous versions when needed.
    • Branches: You can create branches to work on separate features or fixes without disrupting the main code.
    • Commits: Changes are grouped into "commits," which are records of the code's state at a specific moment.
    • Push & Pull: You can "push" your changes to a remote repository and "pull" changes from other team members.
    • History: Git maintains a complete history of changes, allowing you to revert to previous versions when needed.
